# Blueprint-Rwanda

## Project structure
- `frontend/pages/`: current live HTML pages
- `frontend/assets/images/`: active images and gallery assets
- `frontend/assets/js/`: shared browser-side config, API, and auth helpers
- `backend/`: Laravel-oriented backend scaffold and API contract
- `docs/backend/`: architecture, delivery, and operations documentation
- `legacy/snapshots/`: archived earlier front-end and PHP prototypes
