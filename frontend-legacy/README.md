# Frontend Layout

## Directories
- `pages/`: all current live HTML pages
- `assets/images/branding/`: logos and brand graphics
- `assets/images/ui/`: reusable UI imagery
- `assets/images/gallery/`: active AVIF gallery assets
- `assets/images/source-asset/`: preserved duplicate source assets from the old `Asset/` folder
- `assets/js/`: shared front-end helpers

## Notes
- Page filenames were kept stable so cross-page links remain simple.
- Asset filenames used by the live site were normalized to lowercase kebab-case.
- Legacy snapshots were moved out of the live front-end path into `legacy/snapshots/`.
